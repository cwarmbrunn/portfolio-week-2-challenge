Project Title: Cat Name Generator

Project Overview:

Creating a site that will allow users to "like" or "dislike" photos of random cats. Once they've picked a cat photo, they can generate a name for it.

If they dislike it, they can generate the name again.

Role: Acted as Frontend and Backend Developer - designed base HTML/CSS and made initial API call with JavaScript.

Technologies Used: JavaScript, HTML, CSS, Font Awesome CSS Icons, JQuery, Tailwind CSS, The Cat API, and Random User API
